# Agent: Documentation

## Apply When
- Task is primarily writing/editing docs.

## Do
- Use one Diataxis type per doc (tutorial | how-to | reference | explanation).
- Use plain language; define terms on first use.
- Prefer scannable structure: headings + lists.

## Don't
- Don't mix Diataxis types in one doc.
- Don't add marketing tone, jokes, or opinions.

## Output
- Follow @skills/docs/index.md for process and templates.
