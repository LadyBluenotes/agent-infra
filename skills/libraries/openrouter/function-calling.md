# Function Calling

## Setup

- Define your tool schema separately from model logic.

## Core Patterns

- Keep tool inputs minimal and validated.
- Route tool execution server-side.

## Common Mistakes

- Letting the model call tools without validation
- Passing large, unnecessary schemas
