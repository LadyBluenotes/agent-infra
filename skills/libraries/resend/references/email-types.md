# Email Types

## Transactional

Required emails tied to user actions or account status.

Examples:
- Password resets
- Receipts
- Security alerts
- Order confirmations

## Marketing

Promotional content or campaigns.

Examples:
- Newsletters
- Product announcements
- Discounts

## Guidance

- Do not mix promotional content into transactional emails
- Use separate sending subdomains for transactional vs marketing
- Use separate lists and suppression policies
