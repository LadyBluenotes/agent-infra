# Transactional Emails

## Content

- Make the action clear above the fold
- Keep subject lines specific
- Preheader under 90 characters
- Provide expiration or time sensitivity

## Layout

- Mobile-first
- 16px body text minimum
- 44x44px tap targets

## Security

- Include "I didn't request this" guidance for security emails
- Use a real reply-to
- Avoid promotional content

## Resend Behavior

- Allow resend after 60 seconds
- Limit to 3 sends per hour
