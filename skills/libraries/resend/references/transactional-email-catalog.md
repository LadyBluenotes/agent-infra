# Transactional Email Catalog

## Auth

- Email verification
- Password reset
- OTP / 2FA
- Security alerts

## Commerce

- Order confirmation
- Shipping updates
- Delivery confirmation
- Refund confirmation
- Invoice / receipt

## SaaS

- Welcome / onboarding
- Subscription renewal
- Payment failed
- Plan change confirmation

## Guidance

- Send immediately for security and reset emails
- Include expiration times for OTPs and reset links
- Provide support contact for sensitive issues
