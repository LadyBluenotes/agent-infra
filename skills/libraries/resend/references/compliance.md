# Compliance

## CAN-SPAM (US)

- Clear sender identity
- Accurate subject line
- Physical postal address
- Clear unsubscribe link
- Honor opt-out within 10 business days

## GDPR (EU)

- Lawful basis (consent for marketing)
- Record consent (who, when, how)
- Allow access/deletion
- Minimize retention

## CASL (Canada)

- Express consent (or limited implied consent)
- Identify sender and contact info
- Unsubscribe in every marketing email
- Retain consent records for 3 years

## Unsubscribe Requirements

- Provide one-click unsubscribe when possible
- Accept POST for one-click unsubscribe
- Unsubscribe should complete within 48 hours
- Keep unsubscribe simple (no login required)

## Consent Records (Store)

- Email address
- Timestamp
- IP address and user agent
- Source (form name / URL)
- Consent language shown

## Other Regions

- Australia: Spam Act 2003
- UK: PECR + GDPR
- Brazil: LGPD
