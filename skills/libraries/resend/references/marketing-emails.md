# Marketing Emails

## Consent First

- Use explicit opt-in
- Prefer double opt-in for list quality
- Keep proof of consent

## Content Guidelines

- Clear, specific subject lines (<= 50 chars)
- Scannable layout and clear CTA
- Mobile-first design
- Include physical address and unsubscribe link

## Frequency

- Start with lower cadence
- Offer preference center (frequency and topics)

## Required Elements

- Sender identity
- Physical address
- Unsubscribe link (one-click preferred)
