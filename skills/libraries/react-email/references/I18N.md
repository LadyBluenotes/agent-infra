# i18n (Reference)

## next-intl Example

```tsx
import { createTranslator } from 'next-intl';

const messages = {
  subject: 'Welcome',
  body: 'Hi {name}',
};

const t = createTranslator({ locale: 'en', messages });

export function WelcomeEmail({ name }: { name: string }) {
  return (
    <Html lang="en">
      <Head />
      <Body>
        <Text>{t('body', { name })}</Text>
      </Body>
    </Html>
  );
}
```

## Best Practices

- Always pass locale into the template
- Set `Html` `lang` and `dir` for RTL locales
- Translate subject lines
- Provide fallbacks for missing translations
