# Rules: Git

## Apply When
- Any git operation.

## Do
- Avoid destructive/irreversible commands unless explicitly requested.
- Keep commits scoped and descriptive.
- Prefer new commits over amend/rewrite unless asked.

## Don't
- Don't force push unless explicitly requested.
- Don't rewrite published history unless explicitly requested.
- Don't commit files likely containing secrets.
- Don't commit files generated by build scripts.
- Don't commit files generated by tests.
- Don't commit files generated by linters.
- Don't commit files generated by IDEs.
- Don't commit unless asked explicitly.
